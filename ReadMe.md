# envor

Get env vars to types with default.

# Usage

```rust
use envor::envor::*;

fn main(){
	println!("usage {:?}", Foo{});
}
```

# Advanced

```rust
use envor::envor::*;

fn main(){
	println!("advanced {:?}", Foo{});
}
```

# Logging

```bash
export RUST_LOG=info
# or
export RUST_LOG=debug
```
